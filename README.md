# PhotoBook-Project

PhotoBook-Project is a Java Spring application for creating and sharing photo albums.

## Features

    Album creation and deletion
    Photo upload and deletion
    Album and photo viewing
    Search functionality
    Geolocation View
    Mass Image Upload

## Technologies
    Java 15
    HeidiSQL
    Metadata Extractor
    MariaDB
    GeoLocation
